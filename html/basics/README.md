# HTML Basics

Learning basic HTML concepts and practices.

## Files in this folder

### 1. semantic-html.html
Practice using semantic HTML tags:
- `<header>` - page header
- `<nav>` - navigation menu
- `<main>` - main content
- `<section>` - group content
- `<article>` - article content
- `<aside>` - sidebar/related content
- `<footer>` - page footer

What I learned:
- What semantic tags are
- Why to use semantic tags instead of divs
- How to structure a page with semantic tags

### 2. forms-practice.html
Practice creating HTML forms:
- Text input
- Email input
- Textarea for longer text
- Radio buttons for choosing one option
- Select dropdown for options
- Checkbox for multiple selections
- Submit and Reset buttons
- Labels for each input

What I learned:
- How to create form elements
- Input types (text, email, radio, checkbox, etc)
- How to use labels with inputs
- Form structure
