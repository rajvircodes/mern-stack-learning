**What I Learned:**

**Variables**
- `let` - Modern way to declare variables (block scope)
- `const` - Constant variable (cannot change)
- `var` - Old way (avoid using, has function scope)

**Data Types**
- `String` - Text ("Hello", 'World')
- `Number` - Integers and decimals (42, 3.14)
- `Boolean` - true or false
- `Object` - Collection of properties ({name: "Rajvir"})
- `Array` - List of values (["red", "green", "blue"])
- `null` - Intentional empty value
- `undefined` - Variable not defined

**Scope**
- **Global** - Accessible everywhere
- **Function** - Accessible only inside function
- **Block** - Accessible only inside block (if, for, while)
